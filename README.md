# Caesar Cipher Implementation in Java

## Overview
This console-based Java application encrypts and decrypts user data using the classic Caesar Cipher algorithm. It was developed to understand the fundamental concepts of cryptography, data security, and algorithm design.

## Technical Details
* **Language:** Java
* **Concepts Applied:** String manipulation, ASCII character shifting, loops, and conditional statements.
* **Features:** * Encrypts plain text with a custom shift key.
  * Decrypts cipher text back to its original form.
